Browser Fingerprints

November 20th, 2020 by Aayush Kumar

  • Introduction

    A browser fingerprint is a unique ID that is that can be used to identify a particular browser/ computer from the rest of the users. Using this, a lot can be found about specific information like which browser is being used. This is mainly done by the use of the javascript which is used to make the webpages.


    Privacy threat

    For javascript to function, it needs access to certain information like: the resolution of the screen and the orientation of the screen which in specific tells what the height and width is, and what device it is being used on. So javascript can be used to find out a lot of information.

    Few of the many information that can be found about using javascript are which web-browser, operating system, and type of device(laptop, phone) is being used. it can also find out which language you use, your timezone, which browser plug-ins are installed, which specific microsoft apps are installed.

    All these bits of information is taken from us, and is added, to make a unique code that identifies your computer, and is a browser fingerprint.


    Protection

    Browser Fingerprinting invades privacy. It does not matter whether you are logged in, using private browsing mode, or turn on privacy settings, your information will still be tracked. There is an option to turn javascript off, but that's basically breaking 95% of the internet, as every website uses java script,

    Few things that can be done are, try not to be so unique. For example, if use apps, and browsers that is common. Also, don't get unique extensions to your browser. The more uniqueness, makes the Browser fingerprint more unique. Spread awareness about this, by sharing articles like this with others, and educating more people about this topic
